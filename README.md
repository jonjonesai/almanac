# SP2 — A vertical-video content pipeline for small brands

A production-grade pipeline that turns a real-world topic into a 30-second 9:16 vertical video with karaoke captions, ready to post to TikTok / Instagram Reels / Pinterest. Around **$0.33 per video** in API costs.

```
Topic         →  Claude (script)      →  Freepik / FLUX (imagery)
                  ↓
              ElevenLabs (voice)
                  ↓
              WhisperX (word timing)
                  ↓
              MusicGen (background score)
                  ↓
              HyperFrames + GSAP (composition + render)
                  ↓
              Cloudinary → Metricool → social platforms
```

Five example brands are included as reference implementations, each with its own visual identity, brand voice, and topic researcher.

## Why does this exist?

Most social video generators produce content that *looks* AI-made: generic stock footage, hollow narration, hard-sell calls-to-action. The output is technically vertical video, but no human enjoys watching it.

This pipeline is built around an opposite doctrine:

> **Warm hug for the ICP, never a pitch.**

The script's only job is to be a story, a fact, a moment of value the viewer actually wants. The brand sells itself via the outro card + the bio — never via the script body. Two structural patterns enforce this:

1. **`verified_facts` injection** — every numerical claim must be quoted literally from a structured-data source (ephemeris, PubMed paper, release date, cultural calendar). No paraphrasing of dates / counts / effect sizes. Drift goes from "possible" to "impossible without breaking the prompt contract."
2. **Per-beat content contract** — each of the 7 beats has a declared role (hook / tension / reveal / reinforce / surprise / insight / stinger). Beat 7 explicitly forbids product mentions. The contract is in the prompt; deviations earn a re-roll.

The result is content that feels written by someone who actually knows the subject — because every number in it *is* real.

## Requirements

- **Python 3.10+**
- **Node 18+** with `npx hyperframes` (renders the composition)
- **Headless Chromium** (HyperFrames installs this; on Linux: `sudo apt install chromium-browser`)
- **ffmpeg** (for video normalization)
- **Claude Code CLI** — install: https://claude.com/claude-code, then `claude setup-token`
- **API keys** — see [`.env.example`](.env.example) (Anthropic, ElevenLabs, Replicate, Freepik; Cloudinary + Airtable + Metricool optional)

## Quick start

```bash
git clone https://github.com/jonjonesai/sp2-hyperframes.git
cd sp2-hyperframes
cp .env.example .env
# fill in your keys

# install Python deps (just stdlib + skyfield for WLH; everything else is HTTP)
pip install skyfield

# install HyperFrames + render deps
cd welovehoroscope && npx hyperframes init && cd ..

# smoke-test a fire (uses --dryrun by default if HEARTBEAT_DIR/DRYRUN exists)
python3 sp2_pipeline.py --brand welovehoroscope \
  --topic-override "Mars enters Taurus May 19" \
  --angle-override "$(cat docs/sample-brief.txt)" \
  --dryrun
```

`--dryrun` runs everything except the final Metricool scheduling (Cloudinary upload still happens so you can preview the mp4).

## Adding a new brand

1. Copy [`templates/brand.template.json`](templates/brand.template.json) → `<brand>/brand.json` and fill in voice ID, colors, fonts, Airtable IDs, social handles.
2. Copy [`templates/DESIGN.template.md`](templates/DESIGN.template.md) → `<brand>/DESIGN.md` and document the visual identity (colors, motion eases, anti-patterns).
3. Fork the most tonally similar example brand's `scripts/build-karaoke-html.py` into `<brand>/scripts/`. The 5 examples cover: cosmic illustration (`welovehoroscope`), hip-hop punch (`customcreative`), wellness contemplation (`organicaromas`), artisan slow (`utamaspice`), island energy (`taiwanmerch`).
4. Initialize the HyperFrames project: `cd <brand> && npx hyperframes init`.
5. (Optional) Write a topic researcher modeled on the closest existing one. The 5 patterns:
   - `welovehoroscope/scripts/wlh_topic_researcher.py` — structured data source (skyfield ephemeris)
   - `organicaromas/scripts/oa_topic_researcher.py` — API-driven research (PubMed E-utilities)
   - `customcreative/scripts/cc_topic_researcher.py` — fixed calendar (album anniversaries) + Reddit
   - `utamaspice/scripts/ut_topic_researcher.py` — cultural calendar + Reddit
   - `taiwanmerch/scripts/tm_topic_researcher.py` — cultural calendar + Reddit

See [`docs/BRAND_ONBOARDING.md`](docs/BRAND_ONBOARDING.md) for the full checklist.

## The doctrine, in one paragraph

A topic is worth a video only if (a) it's anchored on a real-world fact someone could verify, and (b) it's something an ICP member would feel *informed by*, not *sold to*. The pipeline encodes this as a JSON contract Claude has to obey — `verified_facts` in, scripts out. Beat 7 has no product mention. The brand presence lives in the outro card + the IG bio + the consistent visual identity. People sophisticated enough to be in the ICP will find the products themselves.

Full doctrine in [`docs/DOCTRINE.md`](docs/DOCTRINE.md).
